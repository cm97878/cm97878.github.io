Kitsune

A changelog would probably go here. Or somethin' like that. for now, I'll use it to explain the game instead. The basic premise is that the 'player character' (if you can call it that) is a fox, and the game starts off with it killing creatures and absorbing soul. The progression comes in the form of that soul acting as a multiplier to combat stats, and used to buy upgrades. After enough, another tail is gained, and at 9, you can reset pretty much everything in order to gain the first prestige currency, which unlocks new and more powerful upgrades, so on and so forth.
